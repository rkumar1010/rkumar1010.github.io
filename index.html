Nile River Flow (1871-1970)

This presentation is created for the Coursera course "Developing Data Products" project. The project is done to show that following concepts are understood

  • Shiny to build the application

  • Slidify to create project presentation

The Project

A simple application called Nile River Flow (1871-1970) has been developed and deployed at: https://rkumar1010.shinyapps.io/NileApp/

The application allows the user to:

  1. Select the input - range of years.

  2. Select the output to be displayed on the plot.

Data

This application is using dataset "Nile" which is included with R by default.

The Nile is a major north-flowing river in north-eastern Africa, generally regarded as the longest river in the world. It is 6,853 km (4,258 miles) long. The Nile is an "international" river as its water resources are shared by eleven countries, namely, Tanzania, Uganda, Rwanda, Burundi, Congo-Kinshasa, Kenya, Ethiopia, Eritrea, South Sudan, Sudan and Egypt. In particular, the Nile is the primary water source of Egypt and Sudan.
